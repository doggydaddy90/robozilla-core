# Organizations
#
# This folder contains organization-level configuration for RoboZilla.
#
# Conventions:
# - `orgs/domains/` contains NON-SELLABLE "macro-domain" organizations used only
#   for Flo Prime routing scope.
# - `orgs/packages/` contains sellable organizations and their packaged assets.
#
# See `docs/architecture/DOMAIN_ORGS.md` for the required routing convention.
